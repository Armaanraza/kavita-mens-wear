# Kavita Menswear Premium Website

Production-ready Next.js App Router website for a premium offline-first menswear brand in Gondia. The design direction uses the supplied logo and the provided brand palette with editorial typography and luxury-style layouts.

## Included
- Home, About, Collections, Gallery, Contact, custom 404
- Metadata, Open Graph, schema markup, robots, sitemap
- Floating WhatsApp, click-to-call, Google Maps links
- Mobile-first responsive layout
- Next.js + TypeScript + Tailwind CSS structure

## Run locally
```bash
npm install
npm run dev
```

## Deploy
- Vercel: import the project and deploy directly
- GitHub + Vercel: push this folder to GitHub, then import to Vercel
- GitHub Pages is not ideal for App Router projects unless exported or adapted to static workflow

## Creative direction notes
The information architecture and premium positioning direction borrow from the reference brand patterns of Bombay Shirt Company and Blackberrys, especially around store experience, service-led messaging, fit/fabric emphasis, and premium category storytelling [page:1][page:2].
